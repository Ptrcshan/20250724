# Hope You Like It 💛

Website pesan interaktif dengan animasi partikel bintang.

## Cara Pakai

1. Letakkan file `backsound.mp3` di folder yang sama dengan `index.html`
2. Upload semua file ke hosting kamu
3. Buka website-nya — langsung muncul tulisan pertama!

## Struktur Folder

```
Hope-You-Like-It/
├── index.html
├── backsound.mp3        ← letakkan file lagu di sini
├── styles/
│   └── main.css
└── scripts/
    ├── universe.js
    └── main.js
```

## Fitur

- ✦ Tulisan muncul langsung (~0.8 detik) tanpa menunggu lama
- ✦ Bintang lebih cepat & dinamis
- ✦ Responsif: HP & laptop, portrait & landscape
- ✦ Backsound otomatis setelah tap pertama
- ✦ Tombol mute/unmute musik (pojok kanan atas)
- ✦ Indikator titik progress di bagian atas
- ✦ Hint "TAP UNTUK LANJUT" muncul setelah partikel terbentuk
